# 📊 Sales Dashboard – E-commerce Analysis

## 🎯 Project Overview
This project is an Excel-based sales dashboard designed to analyze business performance, customer behavior, and revenue trends.

---

## 📈 Key Metrics
- Total Revenue
- Total Orders
- Total Customers
- Average Order Value

---

## 📊 Dashboard Features
- Revenue by Category
- Payment Method Analysis
- Top Products by Revenue
- Monthly Sales Trend

---

## 🛠 Tools Used
- Microsoft Excel
- Pivot Tables
- Data Visualization

---

## 💡 Key Insights
- Electronics generated the highest revenue
- Card payments were the most used method
- January recorded higher sales than February
- Top-performing products contributed significantly to total revenue
